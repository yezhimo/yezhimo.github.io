javaScript

  1. 什么是js（JavaScript）

    • 由Netscape公司开发
    • 随Navigator导航者一起发布
    • 介于Java与HTML之间、基于对象和事件驱动的编程语言
    • 可直接运行在Web浏览器中的弱类型脚本语言
    • JavaScript 对大小写是敏感的。

  2. 变量 and 常量

    • 命名规则

      • 不能用数字开头,能够用a-z A-Z _ $

      • 不能使用关键字和保留字

      • js严格区分大小写的。

      • 规范：

        • 命名要有意义
        • 驼峰命名法

    • 声明方式

    • 变量的生存期

      • JavaScript 变量的生命期从它们被声明的时间开始。
      • 全局变量会在页面关闭后被删除。
      • 局部变量会在函数运行以后被删除。

  3. 数据类型

    1. number （数字）

      • Infinity (或 -Infinity) 是 JavaScript 在计算数时超出最大可能数范围时返回的值

      • 方法

        1. isNaN() 确定某个值是否是数

        2. toString() 以字符串返回数值

        3. toExponential(amount) 返回字符串值，它包含已被四舍五入并使用指数计数法的数字

        4. toFixed(amount) 返回字符串值，它包含了指定位数小数的数字

        5. toPrecision() 返回字符串值，它包含了指定长度的数字

        6. valueOf() 以数值返回数值

        7. Number() 返回数字，由其参数转换而来

        8. parseFloat() 解析其参数并返回浮点数。

        9. parseInt() 解析其参数并返回整数。

    2. string (字符串)

      • 单引号(‘’) 或者 双引号(“”) 再或者 反单引号(``) 内的皆为字符串
      • 操作字符串

      1. string.length

        • 获取字符串的长度

      2. string.indexOf('name')

        • 返回字符串中指定文本首次出现的索引,无法设置更强大的搜索值（正则表达式）
        • 如果没找到对应的字符函数返回-1

      3. string.lastIndexOf('name')

        • 返回指定文本在字符串中最后一次出现的索引

      4. string.search('name')

        • 搜索特定值的字符串，并返回匹配的位置,无法设置第二个开始位置参数

      5. string.slice(start, end)

      6. string.substring(start, end)

      7. string.substr(start, length)

      8. string.replace('x','y')

        • 替换 不会改变调用它的字符串。它返回的是新字符串。

      9. string.toLowerCase()

        • 把字符串转换为大写

      10. string.toLowerCase()

        • 把字符串转换为小写

      11. string.concat('','hello')

        • 连接两个或多个字符串

      12. string.trim()

        • 删除字符串两端的空白符

      13. string.charAt()

        • 返回字符串中指定下标（位置）的字符串

      14. string.charCodeAt()

        • 返回字符串中指定索引的字符 unicode 编码

      15. string.split(',')

        • 将字符串转换为数组

      16. fromCharCode()

        • 将 Unicode 转换为字符串

      17. localeCompare

        • 用本地特定的顺序来比较两个字符串

      18. match()

        • 找到一个或多个正则表达式的匹配

    3. Array （数组）

      • 数组用于在单一变量中存储多个值
      • 语法

      1. var arr = []

      2. var arr1 = [1,2,3,4]

      3. var arr2 = new Array()

      • 操作数组

        1. 遍历数组 for循环

        2. 遍历数组 Array.foreach(function)

        3. push() 在数组结尾处）向数组添加一个新的元素

        4. arr[arr.length]= ‘gg’

        5. Array.isArray()

        6. join() 方法也可将所有数组元素结合为一个字符串

        7. pop() 方法从数组中删除最后一个元素

        8. shift() 方法会删除首个数组元素，并把所有其他元素“位移”到更低的索引

        9. unshift() 方法（在开头）向数组添加新元素，并“反向位移”旧元素

        10. delete 会在数组留下未定义的空洞。请使用 pop() 或 shift() 取而代之

        11. splice() 方法可用于向数组添加新项

        12. concat() 方法通过合并（连接）现有数组来创建一个新数组

        13. slice() 方法创建新数组。它不会从源数组中删除任何元素

        14. sort() 方法以字母顺序对数组进行排序

        15. reverse() 方法反转数组中的元素

        16. map() 方法通过对每个数组元素执行函数来创建新数组,不会对没有值的数组元素执行函数,不会更改原始数组

        17. filter() 方法创建一个包含通过测试的数组元素的新数组

  6. 作用域 及 变量提升 及闭包

    • 作用域：

      1. js中的作用域，也叫词法作用域

      2. js中只有函数能够形成一个作用域，所以，又叫函数作用域

      3. 变量的作用范围在书写代码的时候，就定下来了，与代码怎么调用无关

      4. 对于函数来说，创建好了函数以后，函数的作用域就定下来了，也就是作用域链也就定下来了

      5. 也就是说不管在哪个位置调用的函数，都是会回到创建它的环境中去执行！

    • 闭包

      1. 函数内部声明的变量，在函数外部无法访问，这个时候就形成了一个封闭的空间，也就是闭包

      2. 闭包包括了当前函数的函数体和函数所处的环境（作用域）

      3. 闭包其实也就是对作用域的应用

      4. 闭包的模型，也就是函数内部嵌套函数

      5. 闭包的好处：可以包含函数内部私有状态，也就是保护函数内部变量

    • 变量提升

      • js预编译实现过程：
        • 1.js首先扫描var关键字，提前到顶端；
        • 2.然后扫描function定义，提到var之后
        • 3.然后再顺序执行

  11. this的指向

    • 全局作用域中this是window
    • 在事件绑定的函数中的this就是事件发生的时候触发事件的元素
    • 函数执行的时候，函数中的this就是调用函数的主体，没有调用主体，那么this指向是全局 （哪个调用就是哪个）

    • 访问/获取节点

    • 创建节点/属性
      • document.createElement(eName)
        • 创建一个节点

        • 对某个节点创建属性
      • document.createTextNode(text)

    • 添加节点
      • document.insertBefore(newNode,referenceNode)

        • 给某个节点添加子节点
    • 复制节点
      • cloneNode(true | false)

      • parentNode.removeChild(node)
        • 删除某个节点的子节点 node是要删除的节点

      • appendData(data)
        • 将data加到文本节点后面
      • deleteData(start,length)
        • 将从start处删除length个字符
      • insertData(start,data)
        • 在start处插入字符,start的开始值是0;
      • replaceData(start,length,data)
        • 在start处用data替换length个字符
      • splitData(offset)
        • 在offset处分割文本节点
      • substringData(start,length)
        • 从start处提取length个字符
    • 属性操作
      • getAttribute(name)
        • 通过属性名称获取某个节点属性的值
      • setAttribute(name,value)
        • 修改某个节点属性的值
      • removeAttribute(name)
        • 删除某个属性
    • 查找节点
      • parentObj.firstChild
        • 如果节点为已知节点的第一个子节点就可以使用这个方法。此方法可以递归进行使用
      • parentObj.lastChild
        • 获得一个节点的最后一个节点、
      • parentObj.childNodes
        • 获得节点的所有子节点，然后通过循环和索引找到目标节点
    • 获取相邻的节点
      • curtNode.previousSibling
        • 获取已知节点的相邻的上一个节点
      • curtNode.nextSlbling
        • 获取已知节点的下一个节点
    • 获取父节点
      • childNode.parentNode
        • 得到已知节点的父节点
    • 替换节点
      • replace(newNode,oldNode);

  24. 事件冒泡或事件捕获

    • 事件传递定义了元素事件触发的顺序
    • 如果你将

      元素插入到

      元素中，用户点击

      元素, 哪个元素的 “click” 事件先被触发呢？

      • 在 冒泡 中，内部元素的事件会先被触发，然后再触发外部元素，即：

        元素的点击事件先触发，然后会触发

        元素的点击事件。

      • 在 捕获 中，外部元素的事件会先被触发，然后才会触发内部元素的事件，即：
        元素的点击事件先触发 ，然后再触发

        元素的点击事件。

    • addEventListener(event, function, useCapture);
      默认值为 false, 即冒泡传递，当值为 true 时, 事件使用捕获传递。
    • removeEventListener() 方法移除由 addEventListener() 方法添加的事件
