flex布局

display:felx;

  1. 定义容器内元素排列方向 (默认横向排列)

    • flex-direction
      • row;
        • 水平左到右
      • row-reverse
        • 水平右到左
      • column
        • 垂直，上到下
      • column-reverse
        • 垂直，下到上

  2. 定义容器内元素换行 (默认不换行)

    • flex-wrap
      • nowrap
        • 元素不换行
      • wrap
        • 换行
      • wrap-reverse
        -

  3. 元素在主轴（页面）上的排列

    • justify-content
      • center
      • flex-start
        • 左对齐
      • flex-end
        • 右对齐
      • space-between
        • 两端对齐，项目之间间隔相等；
      • space-around
        • 每个元素两侧的间隔相等

  4. 元素垂直对齐

    • align-items

      • flex-start

      • flex-end

      • center

        • 居中

      • baseline

        • 如弹性盒子元素的行内轴与侧轴为同一条，则该值与’flex-start’等效。其它情况下，该值将参与基线对齐

  5. 元素靠边位置 （垂直）

    • align-content
      • flex-start
      • flex-end
      • stretch
        • 元素位于容器的中心
      • center
        • 默认值
        • 元素被拉伸以适应容器
      • space-between
      • space-around

  6. flex-flow属性

    • flex-direction和flex-wrap的简写
    • 默认row nowrap
    • flex-direction
    • flex-wrap
