  10. 开发注意事项

    • 前端布局应适应各主流屏幕 (媒体查询, 百分比)

      • 标签的使用需注意 语义化

    • 需兼容主流浏览器/主流操作系统

      • 浏览器排行 https://www.mydll.com.cn/category-liulanqi-paiming

    • 输入框 类型 type

      • 如一个密码输入框 type 应为 password

    • 需注意 id 及 class 的命名

    • 用户体验

      • 应考虑用户习惯/方便用户进行操作
      • 删除/修改/支付 等操作需进行二次确认

    • 团体开发

      • 统一开发工具 换行方式 编码格式
      • 代码冲突需协商后合并
      • 一小改 一commit
      • id 及 class 命名约定

    • 后端需严格验证前端传值
