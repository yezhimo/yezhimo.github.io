github+hexo搭建博客

  1. 安装 nodejs

  2. 安装 git

  3. 安装Hexo

    • npm i -g hexo-cli

  4. 初始化一个新的博客

    • hexo init

  5. 安装依赖

    • npm i

  6. 开启本地服务器

    • hexo s
    • hexo s -p 5000

  7. 输入 地址查看博客

    • http://127.0.0.1:4000

  8. 写博客

    • 把写好的 markdown 文件，存放到 /source/_posts 文件夹 中即可
    • tips
      • 注意 文件名字不能过长

  9. 配置 NexT 主题（可选）

    • NexT 主题官方文档： [http://theme-next.iissnan.com]

  10. 完成部署GitHub

    • 每个 Github 账户都可以创建一个 YourName.github.io 的仓库，

    • 这个仓库的 master 分支是可以直接使用 YourName.github.io 在网络中访问的

    • 我们可以利用 Github 的这个功能来进行静态页面的展示。

    • 有两种部署的方式，一种是手动部署，一种是使用 hexo 命令行部署

      • 手动部署

        1. 清空之前生成的博客文件
          • hexo clean
          • 如果服务器开启中，可以使用 Ctrl + c 停止
        2. 生成博客文件
          • hexo g
        3. 进入 public 文件夹中，开启 bash 黑窗
          • 输入git init 初始化仓库，并使用 add 和 commit 提交
        4. 使用 git push 命令推送到 Github 仓库
          • git push https://github.com/YourName/YourName.github.io.git master --force
          • 注意后面需要加上 --force 强制覆盖
        5. 在浏览器输入 YourName.github.io 来访问你的博客

      • 使用 hexo 命令部署

        • 此方法需要配置 SSH 密钥来免密提交

        1. 安装 hexo 部署工具

          • npm i hexo-deployer-git --save

        2. 找到 ‘~/blog/_config.yml’ 文件，修改最后的代码：

        3. 清空之前生成的博客文件

          • hexo clean

        4. 生成博客文件

          • hexo g

        5. 部署到 Github

          • hexo d
          • 最后显示 INFO Deploy done: git 代表部署成功

        6. 在浏览器输入 YourName.github.io 来访问你的博客

  11. 总结常用命令

    • hexo clean
      • 清空 (hexo generate) 生成的博客
    • hexo g (hexo generate)
      • 生成博客文件
    • hexo s (hexo server)
      • 启动本地服务器
    • hexo d (hexo deploy)
      • 按照配置好的_config.yml文件，完成部署
    • hexo new “我的新帖子”
      • 创建一个新帖子

  12. 博客加密

    • https://github.com/MikeCoder/hexo-blog-encrypt/blob/master/ReadMe.zh.md

  13. 后台管理

    • https://www.cnblogs.com/xingyunblog/p/8681205.html

  14. 博客添加2d 动画

    1. 安装依赖

      • npm install --save hexo-helper-live2d

    2. 下载动画资源

      • 各种动画model
        • https://github.com/xiazeyu/live2d-widget-models.git
      • 下载后将packages下面的动画模板丢到博客的node_modules里面
      1. 你也可以使用 npm 在博客目录下载单个动画

    3. 修改博客配置文件_config.yml

    4. 重新生成博客

      • 博客页面有你使用的动画，那么恭喜你，成功了
      • 官方链接： https://www.npmjs.com/package/hexo-helper-live2d
