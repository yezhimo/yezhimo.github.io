vue入坑之路

  1. vue介绍

    • 渐进式框架
    • 组件化
    • 官网： https://cn.vuejs.org/
    • github： https://github.com/vuejs/vue

  2. 安装vue-cli

    • 安装node，请看博客 node
    • npm i -g vue-cli
    • 查看版本
      • vue -V

  3. 创建

    • vue init webpack my_project
    • 脚手架分类
      • webpack
        • 基于webpack的开发环境，集成了代码检查、测试、热部署等功能
      • webpack-simple
        • 基于webpack的简单版开发环境
      • browerify
        • 基于browerify的开发环境，集成了代码检查、测试、热部署等功能
      • browerify-simple
        • 基于browerify的简单版开发环境
      • simple
        • 只提供解析.vue文件的普通开发环境
    • 项目结构
      • build
        • webpack相关配置与脚本
      • config
        • 配置文件，主要用于区分开发环境、测试环境、线上环境
      • src
        • 项目源码及引用的资源
      • static
        • 不需要webpack处理的静态资源
      • test
        • 测试脚本

  4. 安装依赖

    • npm i

  5. 运行项目

    • npm run dev

  6. 项目打包

    • npm run build
    • 打包配置
      -

  7. MVVM

    • M
      • Model(数据模型)
    • V
      • View(视图模型,负责将数据模型转化成UI展现出来，就是那些DOM结构)
    • VM
      • ViewModel(一个同步View和Model的对象)

  8. MVC

    • M: Model 数据模型（专门用来操作数据，数据的CRUD）
    • V：View 视图（对于前端来说，就是页面）
    • C：Controller 控制器（是视图和数据模型沟通的桥梁，用于处理业务逻辑）

  9. 指令

    • 插值表达式

    • v-text

      • 将一个变量的值渲染到指定的元素中

    • v-html

      • 被插入的内容都会被当做 HTML

    • v-bind

      • 可以给html元素或者组件动态地绑定一个或多个特性，例如动态绑定style和class
      • 使用方式

    • v-for

    • v-model

      • 默认是string类型
        • 可以通过.number方式确保获取到的是数字类型
      • 在表单控件或者组件上创建双向绑定
      • 支持的元素
        • input
        • select
        • textarea
        • components（Vue中的组件）
      • <input type="text" v-model="uname" />

    • v-on

      • 绑定事件监听，

      • 表达式可以是一个方法的名字或一个内联语句，如果没有修饰符也可以省略，

      • 用在普通的html元素上时，只能监听原生DOM 事件

      • 常用事件

        • click
        • keydown
        • keyup
        • mousedown
        • mouseover
        • submit

      • 阻止默认行为

        • <form v-on:submit.prevent></form>

      • 阻止冒泡

        • .stop

      • 优先触发

        • .capture

      • .self

        • 只有自己能触发，子元素无法触发

      • .once

        • 只能提交一次

      • v-pre

        • 跳过元素的编译，显示原始标签

      • 阻止提交

        • .prevent
          • 只作用于 a 和 form

      • .lazy

        • 相当于监听change操作,失去焦点

      • 按键修饰符

        • <input type="text" @keydown.enter="kd1">
          • 当按下回车键时触发kd1事件
        • enter 回车
        • tab
        • delete 删除/退格
        • esc
        • space
        • up
        • down
        • left
        • right

    • v-if 和 v-show

      • v-if

        • 根据表达式的真假值,是否渲染
        • 在切换时元素及它的数据绑定被销毁并重建

      • v-show

        • 根据表达式的真假值，切换元素的 display CSS 属性

    • v-cloak

      • 实例结束编译后自动移除

    • ref

    • keep-alive

      • 如果把切换出去的组件保留在内存中，可以保留它的状态或避免重新渲染

    • 自定义指令

      • 利用Vue.directive('指令id',{inserted:function(el,binding){}})

  10. 过滤器

    • 私有过滤器

    • 全局过滤器

  11. 计算属性

    • computed
      • 有缓存的

  12. 侦听器

    • watch

  13. 动画

    • transition
      • https://cn.vuejs.org/v2/guide/transitions.html
      • 钩子函数动画
        • before-enter
          • 动画执行前, 用于设置默认值
        • enter
          • 动画执行中, 用于编写动画过程
        • after-enter
          • 动画结束后, 用户重置样式
        • before-leave
          • 动画执行前, 用于设置默认值
        • leave
          • 动画执行中, 用于编写动画过程
        • after-leave
          • 动画结束后, 用户重置样式
    • 帧动画
      • animation
        • https://daneden.github.io/animate.css/

  14. axios

    • 基于 promise 的 HTTP 库

    • get请求示例

    • post请求示例

      • 子传父 （$emit）

    • is属性 + component组件实现动态组件

  15. 路由

    • https://router.vuejs.org/zh/

    • vue-router

      • 路由嵌套

    • 路由钩子

  16. 组件生命周期

    • https://blog.csdn.net/animatecat/article/details/82496801

  17. 代理 proxyTable (config/index.js)

  18. Vuex

    • 状态管理插件 (数据管理)
    • 配置项
      • state
        • 定义所有的状态
      • getter
        • 定义获取状态的方法
        • 如果不需要对状态进行加工，直接通过state访问属性即可
      • mutations
        • 定义修改状态的方法
        • 想要Vue调试工具监听状态的变化过程，必须调用这里的方法进行修改，否则监听不到
      • actions
        • 定义异步调用mutations方法的方法
        • 想要Vue调试工具准确计算状态异步变化后的值，必须在这里异步调用mutations里的方法，否则值有错误
      • modules
        • 分割store为不同的模块
