webpack

  • 介绍

    • JavaScript 应用程序的模块打包器(module bundler)
    • 分析你的项目结构，找到JavaScript模块以及其它的一些浏览器不能直接运行的拓展语言（Sass，TypeScript等），并将其转换和打包为合适的格式供浏览器使用。
    • SPA（single page web application） 单页应用

  • 重点内容

    • 入口(entry)
    • 输出(output)
    • 加载器(loader)
    • 插件(plugins)

  • 安装

    • npm install webpack -g

  • 配置

    • webpack.config.js

    • webpack-dev-server

      • 自动编译刷新浏览器
      • 安装
        • npm init -y
          • 生成package.json项目描述文件
        • npm install webpack-dev-server -g
      • 运行
        • webpack-dev-server --inline

    • 打包less资源

      • npm install less less-loader style-loader css-loader --save-dev

    • 打包sass 资源

      • npm install node-sass sass-loader css-loader style-loader --save-dev

    • 打包url()请求的资源

      • npm install url-loader file-loader --save-dev

    • es6 转 es5

      • npm install babel-core babel-loader babel-preset-es2015 babel-plugin-transform-runtime --save-dev

    • html-webpack-plugin

      • npm i html-webpack-plugin --save-dev

    • html-loader

      • npm i -D html-loader

  • 运行

    • npm run dev
