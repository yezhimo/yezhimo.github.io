  1. 什么是CSS预编译

    • 有人开发了一些扩展CSS功能的写法，比如less,sass,stylus其目的是让css能支持一些编程语言才有的功能

  2. 常见的css 预编译

    • less
      • npm install less -g
    • sass
      • npm install sass -g
    • stylus
      • npm install stylus -g
