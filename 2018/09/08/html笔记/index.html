  4. web标准

    • 结构标准(Structure)：HTML
      • 结构用于对网页元素进行整理和分类，主要包括XML和XHTML两个部分。
    • 样式标准(Presentation)：css
      • 表现用于设置网页元素的版式、颜色、大小等外观样式，主要指的是CSS
    • 行为标准(Behavior)：JavaScript
      • 行为是指网页模型的定义及交互的编写，主要包括DOM和ECMAScript两个部分
    • 好处
      • 让Web的发展前景更广阔
      • 内容能被更广泛的设备访问
      • 更容易被搜寻引擎搜索
      • 降低网站流量费用
      • 使网站更易于维护
      • 提高页面浏览速度

  7. 标签语义化

    • 主要目的就是让大家直观的认识标签(markup)和属性(attribute)的用途和作用
    • 使用有意义的单词来作为标签的名字
    • 核心：在合适的地方使用合适的标签
    • 优点
      • 语义化HTML会使HTML结构变的清晰，有利于维护代码和添加样式
      • 让浏览器或是网络爬虫可以很好地解析，从而更好分析其中的内容
      • 便于团队开发和维护，语义化更具可读性，是下一步吧网页的重要动向，遵循W3C标准的团队都遵循这个标准，可以减少差异化。
    • 语义是否良好?
      • 当我们去掉CSS之后，网页结构依然组织有序，并且有良好的可读性
    • <nav> 表示导航
    • <header> 表示页眉
    • <footer> 表示页脚
    • <section> 表示区块
    • <article> 表示文章 如文章、评论、帖子、博客
    • <aside> 表示侧边栏 如文章的侧栏
    • <figure> 表示媒介内容分组

  1. 标签属性

    • id
      • 作为元素的唯一标识(用于控制样式等)
      • 用于a标签进行锚链接跳转
    • class
      • 用以批量控制样式
    • class和id的命名规则
      • 1.可以包含字母、数字、下划线、短横线
      • 2.不能以数字开头
      • 3.不能以短横线开头
      • 4.不能以单个短横线
      • 5.长名称或词组可以使用中横线来为选择器命名
      • 6.不建议使用“_”下划线来命名CSS选择器。
      • 建议：
        • 1.使用有意义的单词
        • 2.使用驼峰命名法。oneTwo userName
    • style
      • 用以控制单个的元素样式
    • type
      值 描述
      button 定义可点击按钮（多数情况下，用于通过 JavaScript 启动脚本）。
      checkbox 定义复选框。
      file 定义输入字段和 “浏览”按钮，供文件上传。
      hidden 定义隐藏的输入字段。
      image 定义图像形式的提交按钮。
      password 定义密码字段。该字段中的字符被掩码。
      radio 定义单选按钮。
      reset 定义重置按钮。重置按钮会清除表单中的所有数据。
      submit 定义提交按钮。提交按钮会把表单数据发送到服务器。
      text 定义单行的输入字段，用户可在其中输入文本。默认宽度为 20 个字符。
    • disabled
      • disabled 属性规定应该禁用 input 元素。
    • accept
      • accept 属性只能与 <input type="file"> 配合使用。它规定能够通过文件上传进行提交的文件类型。
      • 请避免使用该属性。应该在服务器端验证文件上传。
    • control
      • controls 属性规定浏览器应该为视频提供播放控件。
    • autoplay
      • autoplay 属性规定一旦视频就绪马上开始播放。
    • url
      • 统一资源定位符（或称统一资源定位器/定位地址、URL地址等，英语：Uniform Resource Locator，常缩写为URL），有时也被俗称为网页地址（网址）。如同在网络上的门牌，是因特网上标准的资源的地址（Address）。

  6. Web存储

    • sessionStorage 会话存储
    • localStorage 本地存储
    • 特性
      • 设置、读取方便
      • 容量较大，sessionStorage约5M、localStorage约20M
      • 只能存储字符串，可以将对象JSON.stringify() 编码后存储
    • window.sessionStorage 内存
      • 生命周期为关闭浏览器窗口 数据销毁
      • 在同一个窗口下数据可以共享
    • window.localStorage 硬盘
      • 永久生效，除非手动删除
      • 可以多窗口共享
    • 方法
      • setItem(key, value) 设置存储内容
      • getItem(key) 读取存储内容
      • removeItem(key) 删除键值为key的存储内容
      • clear() 清空所有存储内容

  12. HTML5全局属性

    • data-* 使用 data-* 属性来嵌入自定义数据。
      • 属性名不应该包含任何大写字母，并且在前缀 “data-“ 之后必须有至少一个字符,属性值可以是任意字符串。
    • hidden
      • 它规定元素仍未或不再相关。浏览器不应显示已规定 hidden 属性的元素。hidden 属性也可用于防止用户查看元素，直到匹配某些条件（比如选择了某个复选框）。然后，JavaScript 可以删除 hidden 属性，以使此元素可见。
    • spellcheck
      • 规定是否对元素进行拼写和语法检查。
    • tabindex
      • abindex 属性规定元素的 tab 键控制次序（当 tab 键用于导航时）
      • 以下元素支持 tabindex 属性：<a>, <area>, <button>, <input>, <object>, <select> 以及 <textarea>。
    • contenteditable
      • 规定元素内容是否可编辑。
      • 可继承
